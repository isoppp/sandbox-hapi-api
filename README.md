# sandbox-hapi-api

Learn `Beginning API Development with Node.js` (using hapi v17)

- Add Knex Migrations and Seeds: https://isoppp.com/note/2018-08-15/migration-and-seed-using-knex/

## How to

- install docker
- `yarn run db:run`
- `yarn run db:migrate`
- `yarn run db:seed`
- `yarn run db:start`
